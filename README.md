## 快排几种优化
[参考了这篇文章](https://blog.csdn.net/hacker00011000/article/details/52176100)  
1. 选择基准的方法：a)固定位置；b)随机选取；c)三数取中
2. 当数据长度小于一定数量时，使用直接插入排序
3. 在一次分割结束后，可以把与Key相等的元素聚在一起，继续下次分割时，不用再对与key相等元素分割
4. 优化递归操作，使用尾递归
5. 优化不必要的交换
